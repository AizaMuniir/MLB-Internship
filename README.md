# Student Grading System

## Description
A Python-based student grading system that takes student details, subjects, and marks as input. It calculates subject-wise grades, overall average, and final grade.

## Concepts Used
- Variables
- Data Types
- Lists
- Loops
- Conditional Statements
- Functions

## Grading Criteria
- A: 80 and above
- B: 70 - 79
- C: 60 - 69
- F: Below 60

## How to Run
Run the file:
student_grading_system.py
